Instructional Design Document

Mini-Course

Building User-Centric Websites

Building User-Centric Websites course is designed to introduce learners to the principles and practices of web design. This Mini-Course aims to provide an introduction to responsive web design, equipping learners with the skills to create websites that adapt to different screens and devices.

Learning/Knowledge Gap

One specific learning gap that might be addressed in this course is understanding the importance of accessibility in web design. This includes ensuring that websites are usable and navigable by individuals with disabilities, such as those who use screen readers or have other accessibility needs. Learners may not be familiar with web accessibility, so incorporating this topic into the curriculum would be valuable in creating more inclusive and user-friendly websites.

Target Audience

Demographics

Grade: 7-8
Education: Basic familiarity with using computers and the internet, but no prior formal education in web design.

Background

Learners in grades 7-8 are likely to have some experience using computers and the internet for basic tasks such as browsing, social media, and school assignments. They may have some exposure to simple website navigation but are not expected to have any formal training in web design.

Skills and Dispositions

Skills
  • Basic computer literacy: Familiarity with operating systems (Windows or macOS) and common software and applications (Google Drive, web browsers).
  • Proficiency in using the internet for research and communication.
  • Basic understanding of file management and organization.
Dispositions
  • Curiosity and an interest in digital technology.
  • Eagerness to learn new skills related to web design.

Learners in grades 7-8 are at an age where they are starting to develop a more sophisticated understanding of technology and its applications. They are likely to be motivated by creative projects and interested in how websites work.

Course Type

Step-by-Step Instructional Course

A step-by-step instructional course provides a clear and structured learning path for students. This approach is effective for introducing and reinforcing concepts systematically, ensuring that students build a strong foundation in web design and accessibility.

Course Modality

In-Person Learning

In-person learning offers valuable face-to-face interactions, allowing for immediate feedback, real-time discussions, and hands-on guidance. Given that the target audience is in grades 7-8, having a physical presence can help maintain engagement, foster a sense of community, and provide a controlled environment for practical exercises and collaborative activities.

Course Learning Outcomes

By the end of this course, students will be able to:

  • Understand the fundamental principles of web design, including layout, color schemes, and typography.
  • Demonstrate proficiency in using web design tools and software to build functional and visually appealing web pages.
  • Apply responsive web design techniques to create websites that adapt seamlessly to various screen sizes and devices.
  • Recognize the importance of web accessibility and implement best practices for creating user-friendly, inclusive websites.
  • Collaborate with peers to design and develop a user-centric website, incorporating accessibility features to ensure an inclusive user experience.
  • Evaluate and analyze existing websites for accessibility issues and propose effective solutions for improvement.

Learning Activities

Lectures and Presentations: Deliver engaging lectures and presentations to introduce students to the fundamental principles of web design and the importance of accessibility. Use multimedia elements to enhance understanding.

  • Activity Example: Conduct a lecture on the principles of accessible web design. Include real-world examples of websites with good and poor accessibility, emphasizing the impact on user experience.
